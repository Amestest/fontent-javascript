조건문

            프로그래밍 코드가 해석되는 순서를 제어하는 제어문의 한 종류로
            작성된 조건식에 따라 수행되는 코드를 변경하는 구문

if 문

            
                if(조건식){
                    조건식이 true일 경우 실행할 코드;
                }
            
            * 조건식 : 결과가 true, false 인 식

if - else 문

        
            if(조건식){
                조건식이 true인 경우 수행할 코드;
            }else{
                조건식이 false인 경우 수행할 코드;
            }
        
        *else는 조건식을 작성하는 () 없음
