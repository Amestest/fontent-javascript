함수(function)

        - 목표하는 결과를 만들기 위한 코드의 묶음

        - 코드를 {}(블록) 으로 묶어 작성하고 붙여진 이름을 호출해서 사용--
        
        function 함수명(){
            //함수정의
        }

        함수명(); //함수호출

매개 변수(Parameter), 전달 인자(Arguments)

        매개 변수(Parameter) 중간에 개입해서 값을 변환
        - 함수 선언 시() 안에 작성되는 변수
        -> 전달 인자로부터 받은 값을 저장
        -> 함수 정의 부분에서 사용 가능
        전달 인잠(Arguments)
            - 함수 호출 시 () 안에 작성되는 값
            -> 함수 매개 변수로 전달되는 값
